# 🚀 Gestor de Tareas Pro - Python & AI

Este es mi primer proyecto de desarrollo profesional. Está diseñado aplicando **pensamiento computacional** y gestionado con Git.

## 🧠 Lógica de mi Sistema
He diseñado un algoritmo que organiza mi semana basándose en:
* **Prioridad Profesional**: Las tareas de carrera van arriba y en color claro.
* **Persistencia**: El sistema usa un `timestamp` para saber cuánto tiempo lleva una tarea.
* **Alertas**: Si una tarea profesional lleva más de 7 días "En Progreso", el sistema genera un recordatorio.

## 🛠️ Herramientas utilizadas
- **Lenguaje**: Python 3.x
- **Editor**: VS Code
- **Control de versiones**: Git & GitHub
